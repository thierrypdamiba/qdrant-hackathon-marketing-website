# Quick Setup Guide

## Installation

Run this command to install all dependencies:

```bash
npm install
```

## Development

Start the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## What's Included

✅ **Pixel-Perfect Recreation** - All sections match the original design  
✅ **Fully Responsive** - Works on mobile, tablet, and desktop  
✅ **Interactive Elements** - Working FAQ accordion, hover effects  
✅ **Smooth Animations** - Transitions and hover states  
✅ **Modern Stack** - Next.js 14, TypeScript, Tailwind CSS  

## Sections Implemented

1. **Navigation Bar** - Sticky header with links and register button
2. **Hero Section** - Main headline with event details and CTAs
3. **Stats Section** - Three metric cards with key numbers
4. **Building Section** - Three feature cards explaining the hackathon
5. **Prizes Section** - Prize pool and VC opportunities
6. **Schedule Section** - Complete timeline of events
7. **Hosted By** - Organizer information
8. **Sponsors** - Partner company grid
9. **Judges** - Expert panel with profiles
10. **FAQ** - Accordion-style questions and answers
11. **Footer** - Links and company information

## Customization

All styling is done with Tailwind CSS. Key colors:
- Primary Blue: `#3B5EDB`
- Dark Blue: `#2A4BC7`

To customize, edit:
- `/app/page.tsx` - Main page content
- `/tailwind.config.ts` - Theme configuration
- `/app/globals.css` - Global styles

## Production Build

```bash
npm run build
npm start
```
